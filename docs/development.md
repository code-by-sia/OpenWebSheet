# Development Guide

## Setup

```bash
npm install
npm run serve
```

## Build

```bash
npm run build
```

## Testing

```bash
npm run test:unit
```

## Linting

```bash
npm run lint
```

## Contribution Workflow

1. Create an issue.
2. Create a branch.
3. Implement changes.
4. Run tests and linting.
5. Open a pull request.
