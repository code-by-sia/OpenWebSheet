# Development Guide

## Setup

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Testing

```bash
npm test
```

## Type Checking

```bash
npm run typecheck
```

## Contribution Workflow

1. Create an issue.
2. Create a branch.
3. Implement changes.
4. Run tests and type checking.
5. Open a pull request.
